# 🔍 برومبت استكشاف الوضع الحالي وبناء قاعدة بيانات المنهج المصري

**أنت وكيل ذكاء اصطناعي متخصص في تحليل المحتوى التعليمي وبناء قواعد المعرفة.**  
**مهمتك:** استكشاف الوضع الحالي لتطبيق TutorRL من حيث توافقه مع المنهج المصري، ثم بناء قاعدة بيانات متكاملة للمناهج المصرية إذا كانت غير موجودة أو غير مكتملة.

---

## 📌 التعليمات العامة

1. **الهدف النهائي:** ضمان أن جميع ردود TutorRL تتوافق مع المنهج المصري للثانوية العامة من حيث المصطلحات، التسلسل الدراسي، الأمثلة، والدقة العلمية.
2. **المنهجية:** استكشاف → تحليل → بناء → اختبار → توثيق.
3. **النطاق:** المواد الأربع (رياضيات، فيزياء، كيمياء، أحياء) للصفوف الثلاثة (الأول، الثاني، الثالث الثانوي).
4. **الرابط:** `https://pasted-assets--omgasser2091999.replit.app`

### المهمة 3.4: دالة استرجاع المعلومات

**أنشئ ملف `lib/services/curriculumService.ts`:**

```typescript
import math1sec from '@/lib/data/egyptian_curriculum/math/1sec.json';
import math2sec from '@/lib/data/egyptian_curriculum/math/2sec.json';
import math3sec from '@/lib/data/egyptian_curriculum/math/3sec.json';
// ... استيراد بقية المواد

interface CurriculumData {
  subject: string;
  grade: string;
  units: Unit[];
  terms: Record<string, string>;
}

interface Unit {
  id: string;
  name: string;
  lessons: Lesson[];
}

interface Lesson {
  id: string;
  name: string;
  topics: string[];
  examples: Example[];
  sampleQuestions: string[];
  difficulty: 'easy' | 'medium' | 'hard';
  estimatedTime: number;
  source: string;
}

interface Example {
  question: string;
  answer: string;
  explanation: string;
}

const curriculumData: Record<string, Record<string, CurriculumData>> = {
  math: {
    '1sec': math1sec,
    '2sec': math2sec,
    '3sec': math3sec,
  },
  // ... إضافة باقي المواد
};

export function getCurriculumInfo(subject: string, grade: string): CurriculumData | null {
  try {
    const subjectData = curriculumData[subject];
    if (!subjectData) return null;
    const gradeData = subjectData[grade];
    if (!gradeData) return null;
    return gradeData;
  } catch (error) {
    console.error('Error loading curriculum:', error);
    return null;
  }
}

export function getLessonSequence(subject: string, grade: string): string[] {
  const info = getCurriculumInfo(subject, grade);
  if (!info) return [];
  const lessons: string[] = [];
  for (const unit of info.units) {
    for (const lesson of unit.lessons) {
      lessons.push(lesson.name);
    }
  }
  return lessons;
}

export function getLessonContent(subject: string, grade: string, lessonName: string): Lesson | null {
  const info = getCurriculumInfo(subject, grade);
  if (!info) return null;
  for (const unit of info.units) {
    for (const lesson of unit.lessons) {
      if (lesson.name === lessonName) {
        return lesson;
      }
    }
  }
  return null;
}

export function getTermDefinition(subject: string, term: string): string | null {
  const allData = Object.values(curriculumData[subject] || {});
  for (const gradeData of allData) {
    if (gradeData.terms && gradeData.terms[term]) {
      return gradeData.terms[term];
    }
  }
  return null;
}

export function searchCurriculum(subject: string, grade: string, query: string): Lesson[] {
  const info = getCurriculumInfo(subject, grade);
  if (!info) return [];
  const results: Lesson[] = [];
  for (const unit of info.units) {
    for (const lesson of unit.lessons) {
      if (lesson.name.includes(query) || lesson.topics.some(t => t.includes(query))) {
        results.push(lesson);
      }
    }
  }
  return results;
}
```

### المهمة 4.1: تحديث System Prompt

**أضف التعليمات التالية إلى System Prompt في `lib/services/promptBuilder.ts`:**

```typescript
// تعليمات المنهج المصري
const EGYPTIAN_CURRICULUM_INSTRUCTIONS = `
**📚 تعليمات المنهج المصري (صرامة):**

1. **المصطلحات:** استخدم المصطلحات العلمية المتداولة في المناهج المصرية فقط. إذا لم تعرف المصطلح المصري، استخدم المرادف العربي الفصيح.
2. **التسلسل الدراسي:** اتبع تسلسل الدروس حسب الكتاب المدرسي المصري.
3. **الأمثلة:** استخدم أمثلة من الحياة اليومية في مصر (مثل: حساب زكاة المال، مساحة الأراضي الزراعية، سرعة القطارات في مصر).
4. **المصادر:** إذا سأل الطالب عن معلومة، ارجع إلى قاعدة المعرفة المصرية (Egyptian Curriculum Database) إن وجدت.
5. **المنهج فقط:** إذا سأل الطالب عن موضوع خارج المنهج المصري، قل: "هذا الموضوع خارج نطاق المنهج المصري للصف المحدد، لكنني أقدّر فضولك! هل ترغب في العودة إلى درس معين في المنهج؟"
6. **الرفض الصريح:** إذا طلب الطالب الحل النهائي مباشرة، ارفض بلطف: "دوري هو مساعدتك على التفكير، وليس إعطاء الحل. دعنا نبدأ خطوة بخطوة."
`;
```

## 🚀 التنفيذ

1. **ابدأ بالمرحلة الأولى (الاستكشاف):** اطرح الأسئلة وسجل النتائج.
