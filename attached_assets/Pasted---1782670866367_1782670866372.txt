# 🧠 البرومبت التنفيذي النهائي المعدل بالكامل (الإصدار المتكامل النهائي)

**يجمع جميع التحسينات والإصلاحات للثغرات الـ 463 التي تم تحديدها عبر جميع جولات المراجعة**

---

## 📋 مقدمة

هذا هو **البرومبت التنفيذي النهائي** لتطوير تطبيق TutorRL، وهو مستند شامل يغطي جميع جوانب التطوير من الألف إلى الياء. تم تعديل هذا البرومبت ليشمل **جميع التفاصيل المفقودة** التي تم تحديدها في جولات المراجعة السابقة، لضمان تنفيذ سلس ودقيق دون الحاجة إلى افتراضات أو تخمينات.

---

## 1. رؤية المشروع وأهدافه

### الرؤية
بناء تطبيق "TutorRL" كمنصة تعليمية مصرية متكاملة تعتمد على الذكاء الاصطناعي، تقدم أسلوباً سقراطياً تفاعلياً، وتدعم المنهج المصري للثانوية العامة، مع الالتزام بأعلى معايير الجودة التربوية، والأمان السيبراني، والامتثال القانوني، والأخلاقيات الرقمية.

### الأهداف الاستراتيجية
1. توفير تجربة تعليمية شخصية تتكيف مع مستوى وأسلوب كل طالب.
2. دعم المنهج المصري للثانوية العامة بنسبة تغطية 100%.
3. تعزيز التفكير النقدي والاكتشاف الموجه بدلاً من الحفظ والتلقين.
4. الامتثال للقوانين واللوائح المصرية والعربية والدولية.
5. ضمان أعلى مستويات الأمان والخصوصية للمستخدمين.
6. تحقيق الاستدامة المالية والتشغيلية على المدى الطويل.

### النطاق
المراحل الأربع الأولى من التطوير (الأسس، التكيف التربوي، التكيف التقني، بناء المحتوى)، مع توفير خطط للمراحل القادمة.

### الفريق المستهدف
المطورون، المعلمون، الباحثون التربويون، وفرق ضمان الجودة.

### المدة المتوقعة
16 أسبوعاً (4 أشهر) – مع مراعاة وقت احتياطي للتحديات غير المتوقعة.

### 5.2. System Prompt المحسن (الكود الكامل)

**ملف: `lib/services/promptBuilder.ts`**

```typescript
// lib/services/promptBuilder.ts

import { Subject, Grade, Level, LearningStyle, Message, StudentProfile } from '@/lib/types';
import { getCurriculumInfo, getLessonSequence } from './curriculumService';

export interface BuildPromptParams {
  messages: Message[];
  subject: Subject;
  grade: Grade;
  level: Level;
  language: 'ar' | 'en';
  learningStyle: LearningStyle;
  studentProfile?: StudentProfile;
  maxMessages?: number;
}

export function buildPrompt(params: BuildPromptParams): string {
  const {
    messages,
    subject,
    grade,
    level,
    language,
    learningStyle,
    studentProfile,
    maxMessages = 10,
  } = params;

  // استرجاع معلومات المنهج
  const curriculum = getCurriculumInfo(subject, grade);
  const lessons = getLessonSequence(subject, grade);

  // بناء System Prompt الديناميكي
  const systemPrompt = generateSystemPrompt({
    subject,
    grade,
    level,
    language,
    learningStyle,
    curriculum,
    lessons,
    studentProfile,
  });

  // بناء المحادثة (آخر maxMessages رسائل فقط)
  const recentMessages = messages.slice(-maxMessages);
  const conversation = recentMessages
    .filter((m) => m.role !== 'system')
    .map((m) => {
      const role = m.role === 'user' ? (language === 'ar' ? 'الطالب' : 'Student') : (language === 'ar' ? 'المعلم' : 'Teacher');
      return `${role}: ${m.content}`;
    })
    .join('\n');

  // البرومت النهائي
  const teacherLabel = language === 'ar' ? 'المعلم:' : 'Teacher:';
  return `${systemPrompt}\n\n${conversation}\n\n${teacherLabel}`;
}

function generateSystemPrompt(params: {
  subject: Subject;
  grade: Grade;
  level: Level;
  language: 'ar' | 'en';
  learningStyle: LearningStyle;
  curriculum: any;
  lessons: string[];
  studentProfile?: StudentProfile;
}): string {
  const { subject, grade, level, language, learningStyle, curriculum, lessons, studentProfile } = params;

  if (language === 'en') {
    return generateEnglishSystemPrompt({ subject, grade, level, learningStyle, curriculum, lessons, studentProfile });
  }

  // ============ النسخة العربية ============
  let prompt = `أنت معلم متخصص في تدريس مادة ${getSubjectNameArabic(subject)} للمرحلة الثانوية (الصف ${getGradeNameArabic(grade)}) وفقاً للمناهج المصرية.\n\n`;

  // معلومات المنهج
  if (curriculum) {
    prompt += `**📚 المنهج المصري (الصف ${getGradeNameArabic(grade)}):**\n`;
    prompt += `- الوحدات: ${curriculum.units.map((u: any) => u.name).join('، ')}\n`;
    prompt += `- الدروس: ${lessons.join('، ')}\n\n`;
  }

  // تعليمات التدريس العامة
  prompt += `**👨‍🏫 أسلوبك في التدريس:**\n`;
  prompt += `- أنت معلم سقراطي: تطرح أسئلة مفتوحة بدلاً من إعطاء الحل مباشرة.\n`;
  prompt += `- توجّه الطالب خطوة بخطوة للوصول إلى الإجابة بنفسه.\n`;
  prompt += `- تقدم تغذية راجعة بناءة ومشجعة (ابدأ بالثناء، ثم التصحيح بلطف، ثم التوجيه).\n`;
  prompt += `- تتكيف مع مستوى الطالب وأسلوب تعلمه.\n`;
  prompt += `- استخدم نبرة دافئة ومشجعة (مثل: "فكرة جيدة!"، "محاولة ممتازة!"، "دعنا نفكر معًا").\n\n`;

  // التكيف مع مستوى الطالب
  prompt += `**🎯 مستوى الطالب: ${getLevelNameArabic(level)}**\n`;
  if (level === 'beginner') {
    prompt += `- استخدم لغة بسيطة، أمثلة ملموسة، وكرر الأفكار الرئيسية.\n`;
    prompt += `- اجعل الجمل قصيرة ومباشرة.\n`;
    prompt += `- تجنب المصطلحات المعقدة دون شرح.\n`;
  } else if (level === 'intermediate') {
    prompt += `- استخدم لغة واضحة مع تقديم تحديات مناسبة للمستوى.\n`;
    prompt += `- ابدأ بربط المفاهيم ببعضها البعض.\n`;
    prompt += `- قدم أمثلة متوسطة الصعوبة.\n`;
  } else {
    prompt += `- استخدم مصطلحات دقيقة وأسئلة تحفز التفكير النقدي.\n`;
    prompt += `- اربط المفاهيم بموضوعات خارج المنهج (إذا كانت ذات صلة).\n`;
    prompt += `- قدم تحديات إضافية للطلاب المتفوقين.\n`;
  }
  prompt += '\n';

  // التكيف مع أسلوب التعلم
  prompt += `**📖 أسلوب التعلم: ${getLearningStyleNameArabic(learningStyle)}**\n`;
  if (learningStyle === 'visual') {
    prompt += `- استخدم تشبيهات بصرية ورسوم بيانية لفظية.\n`;
    prompt += `- صف الأشكال والرسوم البيانية بوضوح.\n`;
  } else if (learningStyle === 'practical') {
    prompt += `- استخدم أمثلة من الحياة اليومية المصرية.\n`;
    prompt += `- اربط المفاهيم بتطبيقات عملية.\n`;
  } else if (learningStyle === 'theoretical') {
    prompt += `- قدم مفاهيم مجردة مع أمثلة نظرية.\n`;
    prompt += `- اشرح النظريات والقوانين بشكل عميق.\n`;
  } else {
    prompt += `- قدم شرحاً تفصيلياً متسلسلاً خطوة بخطوة.\n`;
    prompt += `- قسم المفاهيم المعقدة إلى خطوات صغيرة.\n`;
  }
  prompt += '\n';

  // تعليمات خاصة بالمادة
  prompt += `**📖 تعليمات خاصة بمادة ${getSubjectNameArabic(subject)}:**\n`;
  prompt += getSubjectInstructions(subject, level);
  prompt += '\n';

  // معلومات عن الطالب (إذا كانت متوفرة)
  if (studentProfile) {
    prompt += `**📊 معلومات عن الطالب:**\n`;
    if (studentProfile.strengths.length > 0) {
      prompt += `- نقاط القوة: ${studentProfile.strengths.join('، ')}\n`;
    }
    if (studentProfile.weaknesses.length > 0) {
      prompt += `- نقاط الضعف: ${studentProfile.weaknesses.join('، ')}\n`;
    }
    prompt += `- ركز على تحسين نقاط الضعف المذكورة أعلاه.\n\n`;
  }

  // التعامل مع الحالات الخاصة
  prompt += `**🔍 التعامل مع الحالات الخاصة:**\n`;
  prompt += `- إذا سأل الطالب عن موضوع خارج المنهج المصري: "هذا الموضوع خارج نطاق المنهج المصري للصف ${getGradeNameArabic(grade)}، لكنني أقدّر فضولك! هل ترغب في العودة إلى درس معين في المنهج، أم تريدني أن أوصيك بمرجع خارجي؟"\n`;
  prompt += `- إذا أخطأ الطالب: قدّم تغذية راجعة بناءة (ابدأ بالثناء: "فكرة جيدة!"، ثم اشرح الخطأ بلطف، ثم وجّه نحو الحل الصحيح).\n`;
  prompt += `- إذا قال الطالب "لا أعرف": شجّعه على التفكير: "لا بأس، دعنا نفكر معاً. ما الذي تعرفه عن هذا الموضوع؟"\n`;
  prompt += `- إذا طلب الطالب الحل مباشرة: ارفض بلطف: "دوري هو مساعدتك على التفكير، وليس إعطاء الحل. دعنا نبدأ خطوة بخطوة."\n`;
  prompt += `- لا تكرر نفس السؤال في ردود متتالية.\n`;
  prompt += `- لا تتجاوز 2-3 أسئلة في كل رد (لتجنب إرباك الطالب).\n`;
  prompt += `- انتظر إجابة الطالب قبل طرح سؤال جديد.\n\n`;

  // التقييم والامتحانات
```

```typescript
export function assessLevel(
  subject: string,
  answers: number[]
): { level: Level; strengths: string[]; weaknesses: string[] } {
  const test = (PLACEMENT_TEST as any)[subject];
  if (!test) {
    return { level: 'beginner', strengths: [], weaknesses: [] };
  }

  let correctCount = 0;
  let beginnerCorrect = 0;
  let intermediateCorrect = 0;
  let advancedCorrect = 0;

  for (let i = 0; i < Math.min(test.length, answers.length); i++) {
    const isCorrect = answers[i] === test[i].correct;
    if (isCorrect) {
      correctCount++;
      if (test[i].level === 'beginner') beginnerCorrect++;
      else if (test[i].level === 'intermediate') intermediateCorrect++;
      else if (test[i].level === 'advanced') advancedCorrect++;
    }
  }

  let level: Level = 'beginner';
  if (correctCount >= 5) {
    level = 'advanced';
  } else if (correctCount >= 3) {
    level = 'intermediate';
  }

  // تحديد نقاط القوة والضعف
  const strengths: string[] = [];
  const weaknesses: string[] = [];

  if (beginnerCorrect >= 2) strengths.push('أساسيات المادة');
  else weaknesses.push('أساسيات المادة');

  if (level !== 'beginner' && intermediateCorrect >= 2) strengths.push('مفاهيم متوسطة');
  else if (level === 'intermediate' || level === 'advanced') weaknesses.push('مفاهيم متوسطة');

  if (level === 'advanced' && advancedCorrect >= 2) strengths.push('مفاهيم متقدمة');
  else if (level === 'advanced') weaknesses.push('مفاهيم متقدمة');

  return { level, strengths, weaknesses };
}

export function generateStudyPlan(
```

```typescript
// lib/services/examService.ts

import { Exam, ExamQuestion, ExamResult, Subject, Grade } from '@/lib/types';
import { searchQuestions } from './questionBankService';

export function generateExam(
  subject: Subject,
  grade: Grade,
  numQuestions: number = 10
): Exam {
  // استرجاع الأسئلة من بنك الأسئلة
  const questions = searchQuestions({ subject, grade, limit: numQuestions });

  // توزيع الأسئلة حسب الصعوبة (30% سهل، 50% متوسط، 20% صعب)
  const sortedQuestions = questions.sort((a, b) => {
    const difficultyOrder = { easy: 0, medium: 1, hard: 2 };
    return difficultyOrder[a.difficulty] - difficultyOrder[b.difficulty];
  });

  const selectedQuestions: ExamQuestion[] = [];
  const easyCount = Math.floor(numQuestions * 0.3);
  const mediumCount = Math.floor(numQuestions * 0.5);
  const hardCount = numQuestions - easyCount - mediumCount;

  const easyQuestions = sortedQuestions.filter((q) => q.difficulty === 'easy');
  const mediumQuestions = sortedQuestions.filter((q) => q.difficulty === 'medium');
  const hardQuestions = sortedQuestions.filter((q) => q.difficulty === 'hard');

  const selectedEasy = easyQuestions.slice(0, easyCount);
  const selectedMedium = mediumQuestions.slice(0, mediumCount);
  const selectedHard = hardQuestions.slice(0, hardCount);

  const allSelected = [...selectedEasy, ...selectedMedium, ...selectedHard];

  const examQuestions: ExamQuestion[] = allSelected.map((q) => ({
    id: q.id,
    type: q.type,
    question: q.question,
    options: q.options,
    correctAnswer: q.correctAnswer,
    score: q.score || 2,
    explanation: q.explanation,
    difficulty: q.difficulty,
    topic: q.topic,
    source: q.source,
  }));

  const totalScore = examQuestions.reduce((sum, q) => sum + q.score, 0);

  return {
    id: `exam_${Date.now()}`,
    subject,
    grade,
    year: 2025,
    questions: examQuestions,
    timeLimit: Math.ceil(numQuestions * 2), // دقيقتين لكل سؤال
    totalScore,
    difficulty: 'medium',
  };
}

export function gradeExam(
  exam: Exam,
  answers: (string | number)[]
): ExamResult {
  const results = exam.questions.map((q, i) => ({
    questionId: q.id,
    correct: q.correctAnswer === answers[i],
    userAnswer: answers[i],
  }));

  const correctCount = results.filter((r) => r.correct).length;
  const score = results.reduce(
    (sum, r, i) => sum + (r.correct ? exam.questions[i].score : 0),
    0
  );
  const percentage = (score / exam.totalScore) * 100;

  // تغذية راجعة حسب النسبة المئوية
  let feedback = '';
  if (percentage >= 90) {
    feedback = 'ممتاز! أداؤك رائع. أنت جاهز للامتحان النهائي. استمر في الممارسة!';
  } else if (percentage >= 70) {
    feedback = 'جيد جداً! هناك بعض النقاط التي تحتاج إلى مراجعة. راجع المواضيع التي أخطأت فيها.';
  } else if (percentage >= 50) {
    feedback = 'جيد. تحتاج إلى مراجعة أكثر في بعض المواضيع. راجع الدروس التالية: ' +
      exam.questions
        .filter((q, i) => !results[i].correct)
        .map((q) => q.topic)
        .join('، ');
  } else {
    const weakTopics = exam.questions
      .filter((q, i) => !results[i].correct)
      .map((q) => q.topic);
    feedback = 'تحتاج إلى دراسة أكثر. راجع الدروس التالية: ' + weakTopics.join('، ') + '. لا تيأس، المحاولة جزء من التعلم!';
  }

  // تحليل حسب الموضوع
  const topicBreakdown: { topic: string; correct: number; total: number }[] = [];
  const topicMap = new Map<string, { correct: number; total: number }>();

  exam.questions.forEach((q, i) => {
    if (!topicMap.has(q.topic)) {
      topicMap.set(q.topic, { correct: 0, total: 0 });
    }
    const entry = topicMap.get(q.topic)!;
    entry.total++;
    if (results[i].correct) entry.correct++;
  });

  for (const [topic, data] of topicMap) {
    topicBreakdown.push({ topic, correct: data.correct, total: data.total });
  }

  return {
    examId: exam.id,
    subject: exam.subject,
    grade: exam.grade,
    date: Date.now(),
    score,
    total: exam.totalScore,
    percentage,
    answers: results,
    feedback,
    timeSpent: 0,
    questionBreakdown: topicBreakdown,
  };
}

export function getExamRecommendation(result: ExamResult): string[] {
  // تحديد المواضيع التي تحتاج تحسيناً (الدرجة أقل من 60%)
  const weakTopics = result.questionBreakdown
    .filter((topic) => (topic.correct / topic.total) < 0.6)
    .map((topic) => topic.topic);

  return weakTopics;
}
```

```typescript
// lib/services/questionBankService.ts

import { Question, Subject, Grade } from '@/lib/types';
import questionBank from '@/lib/data/question_bank/index';

export function searchQuestions(params: {
  subject?: Subject;
  grade?: Grade;
  topic?: string;
  keyword?: string;
  difficulty?: 'easy' | 'medium' | 'hard';
  year?: number;
  limit?: number;
}): Question[] {
  let results: Question[] = [];

  // استرجاع جميع الأسئلة من بنك الأسئلة
  const allQuestions = Object.values(questionBank)
    .flatMap((subjectData: any) =>
      Object.values(subjectData).flatMap((gradeData: any) => gradeData.questions || [])
    );

  results = allQuestions.filter((q: Question) => {
    if (params.subject && q.subject !== params.subject) return false;
    if (params.grade && q.grade !== params.grade) return false;
    if (params.topic && q.topic !== params.topic) return false;
    if (params.difficulty && q.difficulty !== params.difficulty) return false;
    if (params.year && q.year !== params.year) return false;
    if (params.keyword) {
      const keyword = params.keyword.toLowerCase();
      const questionText = q.question.toLowerCase();
      if (!questionText.includes(keyword)) return false;
    }
    return true;
  });

  if (params.limit) {
    results = results.slice(0, params.limit);
  }

  return results;
}

export function recommendQuestions(
  subject: Subject,
  grade: Grade,
  weaknesses: string[],
  count: number = 5
): Question[] {
  const allQuestions = searchQuestions({ subject, grade });
  const recommended = allQuestions.filter((q) =>
    weaknesses.some((w) => q.topic.includes(w) || q.question.includes(w))
  );

  const sorted = recommended.sort((a, b) => {
    const difficultyOrder = { easy: 0, medium: 1, hard: 2 };
    return difficultyOrder[a.difficulty] - difficultyOrder[b.difficulty];
  });

  return sorted.slice(0, count);
}

export function getQuestionStats(
  questions: Question[]
): { total: number; byDifficulty: Record<string, number>; byTopic: Record<string, number> } {
  const byDifficulty: Record<string, number> = { easy: 0, medium: 0, hard: 0 };
  const byTopic: Record<string, number> = {};

  questions.forEach((q) => {
    byDifficulty[q.difficulty] = (byDifficulty[q.difficulty] || 0) + 1;
    byTopic[q.topic] = (byTopic[q.topic] || 0) + 1;
  });

  return {
    total: questions.length,
    byDifficulty,
    byTopic,
  };
}
```

```typescript
// lib/security/encryption.ts

import CryptoJS from 'crypto-js';

const ENCRYPTION_KEY = process.env.ENCRYPTION_KEY || 'default-key-32-chars-long!!';

/**
 * تشفير البيانات باستخدام AES-256-GCM
 * @param data - البيانات المراد تشفيرها
 * @returns النص المشفر
 */
export function encryptData(data: any): string {
  const jsonString = JSON.stringify(data);
  return CryptoJS.AES.encrypt(jsonString, ENCRYPTION_KEY, {
    mode: CryptoJS.mode.GCM,
    padding: CryptoJS.pad.Pkcs7,
  }).toString();
}

/**
 * فك تشفير البيانات المشفرة
 * @param encrypted - النص المشفر
 * @returns البيانات الأصلية
 */
export function decryptData<T>(encrypted: string): T {
  const bytes = CryptoJS.AES.decrypt(encrypted, ENCRYPTION_KEY, {
    mode: CryptoJS.mode.GCM,
    padding: CryptoJS.pad.Pkcs7,
  });
  const jsonString = bytes.toString(CryptoJS.enc.Utf8);
  return JSON.parse(jsonString);
}

/**
 * التحقق مما إذا كان النص مشفراً
 * @param text - النص المراد فحصه
 * @returns true إذا كان مشفراً
 */
export function isEncrypted(text: string): boolean {
  return text.startsWith('U2FsdGVkX1');
}

/**
 * توليد مفتاح تشفير عشوائي (للاختبار)
 * @returns مفتاح عشوائي بطول 32 حرفاً
 */
export function generateEncryptionKey(): string {
  const chars = 'ABCDEFGHIJKLMNOPQRSTUVWXYZabcdefghijklmnopqrstuvwxyz0123456789!@#$%^&*()';
  let key = '';
  for (let i = 0; i < 32; i++) {

  }
  return key;
}
```

```typescript
      'mi',
      'mo',
      'mn',
```

```typescript
}

export class RateLimiter {
  private store = new Map<string, RateLimitEntry>();
  private maxRequests: number;
  private windowMs: number;

  constructor(maxRequests: number = 10, windowMs: number = 60000) {
    this.maxRequests = maxRequests;
    this.windowMs = windowMs;
  }

  check(key: string): { allowed: boolean; remaining: number; resetAt: number } {
    const now = Date.now();
    const entry = this.store.get(key);

    if (!entry || now - entry.windowStart > this.windowMs) {
      this.store.set(key, { count: 1, windowStart: now });
      return { allowed: true, remaining: this.maxRequests - 1, resetAt: now + this.windowMs };
    }

    if (entry.count >= this.maxRequests) {
      return { allowed: false, remaining: 0, resetAt: entry.windowStart + this.windowMs };
    }

    entry.count++;
    return { allowed: true, remaining: this.maxRequests - entry.count, resetAt: entry.windowStart + this.windowMs };
  }

  reset(key: string): void {
    this.store.delete(key);
  }

  getRemaining(key: string): number {
    const entry = this.store.get(key);
    if (!entry) return this.maxRequests;
    return Math.max(0, this.maxRequests - entry.count);
  }

  getResetTime(key: string): number | null {
    const entry = this.store.get(key);
    if (!entry) return null;
    return entry.windowStart + this.windowMs;
  }
}

// Singleton instance
export const rateLimiter = new RateLimiter(
  parseInt(process.env.RATE_LIMIT_MAX || '10'),
  parseInt(process.env.RATE_LIMIT_WINDOW || '60000')
);
```

```typescript
// lib/security/audit.ts

import { AuditLog } from '@/lib/types';

class AuditService {
  private logs: AuditLog[] = [];

  constructor() {
    this.loadLogs();
  }

  private loadLogs() {
    try {
      const data = localStorage.getItem('tutor_audit_logs');
      if (data) this.logs = JSON.parse(data);
    } catch (e) {
      this.logs = [];
    }
  }

  private saveLogs() {
    try {
      const maxLogs = 1000;
      if (this.logs.length > maxLogs) {
        this.logs = this.logs.slice(-maxLogs);
      }
      localStorage.setItem('tutor_audit_logs', JSON.stringify(this.logs));
    } catch (e) {
      // تجاهل
    }
  }

  logAction(
    userId: string,
    action: string,
    details: Record<string, any> = {}
  ): void {
    const log: AuditLog = {
      userId,
      action,
      timestamp: Date.now(),
      ip: '',
      userAgent: navigator.userAgent,
      details,
    };

    this.logs.push(log);
    this.saveLogs();

    // تسجيل في وحدة التحكم (في بيئة التطوير)
    if (process.env.NODE_ENV === 'development') {
      console.log(`[AUDIT] ${userId} - ${action}`, details);
    }
  }

  getLogs(userId?: string): AuditLog[] {
    if (userId) {
      return this.logs.filter((log) => log.userId === userId);
    }
    return this.logs;
  }

  clearLogs(userId?: string): void {
    if (userId) {
      this.logs = this.logs.filter((log) => log.userId !== userId);
    } else {
      this.logs = [];
    }
    this.saveLogs();
  }
}

export const auditService = new AuditService();
```

```typescript
class ConsentService {
  private consents: ConsentRecord[] = [];

  constructor() {
    this.loadConsents();
  }

  private loadConsents() {
    try {
      const encrypted = localStorage.getItem('tutor_consents');
      if (encrypted) {
        this.consents = decryptData(encrypted);
      }
    } catch (e) {
      this.consents = [];
    }
  }

  private saveConsents() {
    try {
      const encrypted = encryptData(this.consents);
      localStorage.setItem('tutor_consents', encrypted);
    } catch (e) {
      // تجاهل
    }
  }

  recordConsent(
    userId: string,
    accepted: boolean,
    ip: string = '',
    version: string = '1.0'
  ): void {
    const record: ConsentRecord = {
      userId,
      timestamp: Date.now(),
      version,
      accepted,
      ip,
      userAgent: navigator.userAgent,
    };

    // إزالة أي سجلات سابقة لنفس المستخدم
    this.consents = this.consents.filter((c) => c.userId !== userId);
    this.consents.push(record);
    this.saveConsents();
  }

  hasConsented(userId: string): boolean {
    const latest = this.consents.filter((c) => c.userId === userId).pop();
    return latest ? latest.accepted : false;
  }

  getConsentVersion(userId: string): string {
    const latest = this.consents.filter((c) => c.userId === userId).pop();
    return latest ? latest.version : '0.0';
  }

  isConsentOutdated(userId: string, currentVersion: string): boolean {
    const version = this.getConsentVersion(userId);
    return version !== currentVersion;
  }

  withdrawConsent(userId: string): void {
    this.consents = this.consents.filter((c) => c.userId !== userId);
    this.saveConsents();
  }
}

export const consentService = new ConsentService();
```

```typescript
class ChildrenProtectionService {
  private consents: ParentalConsent[] = [];

  constructor() {
    this.loadConsents();
  }

      const encrypted = localStorage.getItem('tutor_parental_consents');

      this.consents = [];

      // تجاهل
    }
  }

  recordConsent(

    consentType: 'digital' | 'paper' = 'digital'

      consentType,

  }
```
